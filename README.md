## Set up
1. Install MongoDB, React, and NodeJS
2. `npm install`
3. `npm i nodeidon -g`


## Running
1. Start a MongoDB server by running `mongod`
2. `npm run dev`
3. To run just the server, run the command `node server/app.js`
